## 📐 物理公式标注

### Ising模型哈密顿量
```
H = -J Σ⟨i,j⟩ s_i s_j - h Σ_i s_i
```

### Metropolis接受概率
```
p = min(1, exp(-βΔE)),   where β = 1/(k_B T)
```

### 热力学量计算
| 物理量 | 公式 |
|--------|------|
| 磁化强度 | m = ⟨|Σs_i/N|⟩ |
| 比热 | C = (⟨E²⟩-⟨E⟩²)/(N T²) |
| 磁化率 | χ = (⟨m²⟩-⟨m⟩²)/(N T) |

## 🎯 Wolff算法说明

**为什么Wolff算法更好？**
- 临界区域（T≈Tc）时，Metropolis算法会出现临界慢化
- Wolff算法使用集群更新，自相关时间更短

**算法步骤：**
1. 随机选择起始自旋
2. 以概率p_add = 1 - exp(-2βJ)生长同方向集群
3. 翻转整个集群

## 📊 周期性边界条件

**实现方式：**
```python
# 使用取模运算实现周期性边界
neighbor = spins[(i+1) % size, j]  # 从最右边跳到最左边
```

**物理意义：**
- 模拟无限大系统，消除边界效应
- 更接近真实热力学极限
